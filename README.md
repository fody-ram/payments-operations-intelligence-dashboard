# Payments Operations Intelligence Dashboard

**Built by:** Fadwa Ramadan Ali Hassan  
**Date:** May 2026

### Project Overview

This project simulates real **Operations Intelligence** work at a payments company. It focuses on defining key metrics, merchant cohort analysis, data quality, and delivering actionable insights to leadership.

### Key Deliverables

- Complete **Metrics Dictionary** with 8+ core Ops KPIs
- Interactive Power BI Dashboard (4 pages)
- Merchant Cohort Analysis & Segmentation
- Data processing and KPI calculations using Python

### Tech Stack

- **Python** – Data cleaning, feature engineering & KPI logic
- **Power BI** – Advanced visualizations and DAX
- **Dataset**: Synthetic financial transactions (PaySim)

### Key Insights

- Overall Acceptance Rate: **99.87%**
- TRANSFER transactions have the highest chargeback rate (0.77%)
- Early merchant cohorts show stronger and more stable performance
- Clear patterns in daily volume and risk

### Skills Demonstrated

- Defining and documenting KPIs (Metrics Dictionary)
- Cohort analysis and merchant segmentation
- Building actionable executive dashboards
- Data quality awareness and business translation
